# ImgView

A simple image viewer

<p align="center">
    <img src="./icon.gif" title="ImgView" alt="ImgView"/>
</p>

## Run

```bash
$ cargo build --relase
```

```bash
$ imgview -h
imgview 0.1.0
Ahmed Ibrahim
An image viewer

USAGE:
    imgview.exe [OPTIONS]

OPTIONS:
    -h, --help             Print help information
    -i, --image <IMAGE>    Image path [default: grades.png]
    -V, --version          Print version information
(base)
```

imgvew -i example.png
